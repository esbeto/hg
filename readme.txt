# Introducción #
Hansel & Gretel Sistema Educativo de Apoyo

# Requerimientos del Sistema #
* Servidor web que pueda ejecutar PHP
* Una conexión a una base de datos MySQL protegida.
* PHP 5.3 o mayor

# Instalación #
1. Subir el directorio a un web server.
2. Crear una nueva base de datos.
3. Ingresar al sitio donde se registró para seguir el proceso de instalación.
4. Llenar la forma, hacer click en instalar.
